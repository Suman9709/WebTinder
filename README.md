❤️ Tinder Clone Web Application
A modern full-stack Tinder-style dating web application where users can create profiles, next/back with other users, and chat in real time.


🚀 Features
🔐 User Authentication (Login / Register)
👤 User Profile Creation & Editing
📸 Upload Profile Images
🤝 Match System
💬 Real-Time Chat Between Matched Users
🔔 Notifications
🌐 Responsive Design
🔒 Protected Routes & JWT Authentication


🛠️ Tech Stack


Frontend
React.js
Tailwind CSS
Axios
React Router DOM
TanStack Query (React Query)



Backend
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
Socket.IO
Other Tools
Cloudinary (Image Uploads)
bcrypt.js (Password Hashing)
dotenv
cors

⚙️ Installation
1️⃣ Clone the Repository
git clone https://github.com/your-username/tinder-clone.git
2️⃣ Install Dependencies
Frontend
cd client
npm install


Backend
cd server
npm install

Environment Variables
Create a .env file inside the server folder.


MONGO_URI=
JWT_SECRET_KEY =
JWT_SECRET_KEY_EXPIRY=
CORS_ORIGIN=
